# Restaurant App | Restoran Uygulaması 🍽️

## 🇬🇧 Description
A simple restaurant application built with React Native and Expo. Users can view restaurant menus a modern interface.

### Tech Stack
- React Native
- Expo
- Axios

### Quick Start
```bash
npm install
npx expo start
```
